# Hilal Ahammed PP | Developer Portfolio

A minimal, high-performance personal portfolio built to showcase my work as a Full-Stack MERN Developer. Designed with a focus on fluid micro-animations, clean typography, and a seamless user experience.

🔗 **[Live Preview](https://portfolio-one-gold-29.vercel.app/)**

## ⚡ Tech Stack

- **Frontend:** React.js (Vite), Tailwind CSS
- **Animations:** Framer Motion, Custom CSS Micro-interactions
- **Deployment:** Vercel

## 🚀 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/hilalahmd/portfolio-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
